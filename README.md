# airline-sentiment-analysis
End-to-end NLP project analyzing 14,640 airline tweets using RoBERTa, BERTopic, XGBoost, and Streamlit.

## Tech Stack
Python | HuggingFace Transformers | BERTopic | XGBoost | Streamlit | FastAPI | AWS S3

## Project Status
Week 1 — EDA in progress

## Structure
- notebooks/ — Jupyter notebooks for each project phase
- src/ — reusable Python modules
- outputs/ — saved charts and reports
- app/ — Streamlit dashboard (Week 3)
